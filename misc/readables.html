Technical books and readables I've read

These are books and readables I’ve read during my software development career. If you’re a developer too, feel free to use this as a reading plan.

Although I put Amazon links for these books, I am not affiliated either with Amazon, nor the authors of these books.

Foundational

  • 97 Things Every Programmer Should Know: Collective Wisdom from the Experts (1st Edition)
  • Code Complete: A Practical Handbook of Software Construction (2nd Edition)
  • The Pragmatic Programmer: From Journeyman to Master (1st Edition)

Clean coding, refactoring

  • Clean Code: A Handbook of Agile Software Craftsmanship (1st Edition)

Career

  • The Clean Coder: A Code of Conduct for Professional Programmers (1st Edition)
  • The Passionate Programmer: Creating a Remarkable Career in Software Development (2nd Edition)

Interviewing

  • Cracking the Coding Interview: 189 Programming Questions and Solutions (6th Edition)

Java

Design patterns

  • Design Patterns: Elements of Reusable Object-Oriented Software (1st Edition)

C++

  • API Design for C++ (1st Edition)
  • C++ Coding Standards: 101 Rules, Guidelines, and Best Practices (1st Edition)

Functional programming

  • Becoming Functional: Steps for Transforming Into a Functional Programmer (1st Edition)
